# Emacs Config

My [Emacs](https://www.gnu.org/software/emacs/) config.

![Screenshot of editor](./docs/screenshot.png)

## Installation

Clone this repo into `~/.config/emacs`:

```
git clone https://github.com/unindented/emacs-config.git ~/.config/emacs
```
